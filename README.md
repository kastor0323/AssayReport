# AssayReport

**AssayReport**는 분석 결과를 효율적으로 관리하고 보고서를 생성하는 통합 플랫폼입니다.

## 📋 프로젝트 개요

이 프로젝트는 Python 기반의 백엔드와 React 기반의 프론트엔드로 구성된 풀스택 애플리케이션입니다. 분석 데이터를 수집, 처리, 시각화하고 전문적인 보고서를 생성할 수 있습니다.

## 🛠️ 기술 스택

### 백엔드
- **Python** - 핵심 언어
- 데이터 처리 및 API 개발

### 프론트엔드
- **React** - UI/UX 구현
- **Node.js** - 개발 환경 및 빌드

## 📁 프로젝트 구조

```
AssayReport/
├── Front/                  # React 기반 프론트엔드
│   ├── src/               # 소스 코드
│   ├── public/            # 정적 자산
│   └── package.json       # npm 의존성 관리
├── Back/                  # Python 기반 백엔드
├── docs/                  # 문서
└── README.md
```

## 🚀 시작하기

### 필수 요구사항
- Python 3.8 이상
- Node.js 14.0 이상
- npm 또는 yarn

### 설치

#### 프론트엔드 설치
```bash
cd Front
npm install
```

#### 백엔드 설치
```bash
cd Back
pip install -r requirements.txt
```

### 실행

#### 프론트엔드 실행
```bash
cd Front
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하세요.

#### 백엔드 실행
```bash
cd Back
python app.py  # 또는 해당 시작 명령어
```

## 📚 주요 기능

- ✅ 분석 데이터 관리
- ✅ 실시간 데이터 시각화
- ✅ 자동 보고서 생성
- ✅ 사용자 친화적 대시보드
